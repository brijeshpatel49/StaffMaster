# StaffMaster – Employee Management System (EMS)

StaffMaster is a full-stack Employee Management System built using the MERN stack (MongoDB, Express.js, React.js, Node.js).  
This project is developed as an internship project to manage employee data, attendance, payroll, and performance in a centralized system.

---

## 🚀 Tech Stack

**Frontend:**
- React.js
- Tailwind CSS / CSS

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB (Mongoose)

**Authentication & Security:**
- JWT (JSON Web Token)
- bcrypt for password hashing
- Role-based authorization

---

## 👥 User Roles

- **Admin**
- **HR**
- **Employee**

Each role has specific permissions and access control.

---

## 📌 Core Features

### 🔐 Authentication
- Role-based login system
- Secure password hashing
- JWT-based authentication
- Protected API routes

### 🛠 Admin Panel
- Manage HR accounts
- Manage departments & roles
- View system analytics
- Activate / deactivate users

### 👨‍💼 HR Panel
- Add / edit / delete employees
- Manage attendance
- Approve / reject leave requests
- Generate payroll
- Evaluate employee performance

### 👨‍💻 Employee Panel
- View & update profile
- Mark attendance
- Apply for leave
- View salary & payslips
- View performance feedback
